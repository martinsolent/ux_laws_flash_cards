# UX Laws Flash Cards

An interactive study tool for learning fundamental UX design principles through flip cards. Each card presents a UX law with AI-generated explanations and real UI examples.

## Features

- **Interactive Flash Cards**: Tap any card to flip and reveal detailed explanations
- **Explanations**: Each law includes context with real UI examples
- **Category Filtering**: Filter cards by topic:
  - Heuristics & Interaction
  - Gestalt Principles
  - Cognitive Bias
  - Additional Principles
- **Dark/Light Mode**: Toggle between theme modes for comfortable studying
- **Progress Tracking**: Visual progress bar showing how many cards you've reviewed (20 total)
- **Illustrated Cards**: Each law features a custom SVG illustration for visual reinforcement
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Study Structure

The flash card deck covers 20 essential UX laws organized by principle type. Cards display the law name, short description, and illustration on the front, with full explanations on the back.
